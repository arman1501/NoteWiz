# NoteWiz App

A premium modern note-taking application built with Flutter & Supabase featuring cloud sync, authentication, offline support, beautiful dark UI, and smart note management.

---

## 📱 App Overview

NoteWiz is a professional smartphone application development project designed to provide users with a fast, secure, and elegant note-taking experience.

The app combines modern UI/UX with powerful features like authentication, cloud synchronisation, favourites, trash management, categories, search filtering, and offline functionality.

---

# ✨ Features

- 🔐 Authentication System
- ☁️ Supabase Cloud Integration
- 📝 Create, Edit & Delete Notes
- ⭐ Favourite Notes
- 🗑 Trash Management
- 🔍 Smart Search & Sorting
- 📂 Category Filtering
- 🌙 Dark / Light Theme
- 📶 Offline Mode Support
- 📊 Live Statistics Dashboard
- 💾 Persistent Local Storage
- 🔄 Real-time UI Updates
- 🎨 Modern Premium UI Design
